# Documentation

These docs explain how to use `repo-context-mcp-server` and why it is built the
way it is. The top-level [`README.md`](../README.md) is the reference (tools,
config, transport, data flow); the pages here are the guide.

| Page | What it covers |
|---|---|
| [Getting started](./getting-started.md) | Install, configure, connect to Claude Code, and a tool-by-tool playbook for *when* to reach for each tool. |
| [Why semantic-driven context](./why-semantic-driven-context.md) | The thesis: why pairing meaning-first retrieval with a verified structural graph builds a better context graph than grep/glob or pure-LLM guessing — and why that makes coding agents measurably more reliable. |

If you are evaluating the project, read [Why semantic-driven
context](./why-semantic-driven-context.md) first. If you just want it running,
start with [Getting started](./getting-started.md).
